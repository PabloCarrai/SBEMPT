# SBEMPT

Script en bash para enviar un mensaje a un bot de telegram. El mensaje es iniciando o finalizando. Esto tiene un mensaje seteado dentro del script. Es basico pero es lo que necesito en estos casos.  

Forma de uso:
Configura el archivo config.env-Ejemplo con los datos necesarios y renombralo a config.env. Luego ejecuta 

bash EnviarMensaje.sh Iniciando<br>
bash EnviarMensaje.sh Finalizando
